# Chipatara API

> Backend API for Chipatara application, which is a clinic management web system

## Usage

Rename "config/example.config.env" to "config/config.env" and update the values/settings to your own

## Install Dependencies

```
npm instal
```

## Run App

```
#   Run in dev mode
npm run dev

#   Run in prod mode
npm start
```

- Version: 1.0.0
